홈 화면 SVG 에셋 목록

- icon_signal / icon_wifi / icon_battery: 상태바
- icon_flame: 연속 기록 뱃지
- icon_butterfly: AI 카드 썸네일
- icon_chat: AI 채팅 버튼 및 중앙 탭
- icon_home / icon_calendar / icon_record / icon_user: 하단 탭바
- icon_check: 완료 체크
- icon_plus: 할 일 추가
- event_bar_orange / event_bar_purple: 일정 구분 바
- background_home_gradient: 화면 배경

SVG는 currentColor를 사용한 아이콘이 있어 앱에서 색상 변경이 쉽도록 구성했습니다.
